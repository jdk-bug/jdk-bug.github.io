# Getting Started with the HTML5 Skateboard Game

The **HTML5 Skateboard Game** is an open-source browser game inspired by classic skateboarding titles.

## Features

- **THPS-style controls** — Flip tricks, grab tricks, and grinds using intuitive keyboard shortcuts
- **Responsive design** — Works on desktop and mobile devices
- **Dynamic audio** — Sound effects and background music that react to gameplay
- **Multiple game states** — Main menu, gameplay, and pause screen

## Controls

| Action | Key |
|--------|-----|
| Move | Arrow Keys |
| Jump | Space |
| Flip Trick | Shift + Arrow |
| Grab Trick | C + Arrow |
| Grind | Z |
| Pause | P |
