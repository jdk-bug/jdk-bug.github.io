# Welcome to the Blog

This is the first post on the new blog section of the jdk-bug homepage.

## What to Expect

The blog is built with **vanilla JavaScript** and renders Markdown files directly in the browser. Future posts will cover:

- Project updates
- Technical notes
- Open source work
- Tutorials and guides

## How It Works

Posts are stored as Markdown files in the `/posts/` directory with YAML frontmatter for metadata. A lightweight Markdown parser renders them client-side, and the theme matches the rest of the site.

Stay tuned for more content!
