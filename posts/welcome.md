# Welcome to the Blog

This is the first post on the new **Markdown-powered** blog section of the jdk-bug homepage.

## What to Expect

Future posts will cover:

- Project updates and releases
- Technical deep-dives and tutorials
- Open source contributions and notes
- Behind-the-scenes development stories

## How It Works

The blog is built with **zero build tools** — pure client-side rendering:

1. Posts are stored as Markdown files in the repo
2. A lightweight JavaScript Markdown parser renders them in the browser
3. YAML frontmatter provides metadata like title, date, and tags

No static site generators, no npm, no bundlers. Just vanilla JS and a CDN.

Stay tuned for more posts!
