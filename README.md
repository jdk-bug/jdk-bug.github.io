# jdk-bug.github.io

Personal homepage for [jdk-bug](https://github.com/jdk-bug), built with modern HTML, CSS, and JavaScript — hosted on GitHub Pages.

## ✨ Features

- 🌙 **Dark Mode Toggle** — persistent theme preference via `localStorage`, respects `prefers-color-scheme`
- 🎨 **Modern Design** — Google Fonts (Inter), smooth animations, gradient hero section
- 📱 **Fully Responsive** — optimized for mobile down to 320px
- ♿ **Accessible** — semantic HTML, ARIA labels, `prefers-reduced-motion` support
- 🚀 **Zero Dependencies** — pure HTML/CSS/JS, no frameworks or build tools

## 🚀 Quick Start

Visit the live site: **[jdk-bug.github.io](https://jdk-bug.github.io)**

Or run locally:
```bash
git clone https://github.com/jdk-bug/jdk-bug.github.io.git
cd jdk-bug.github.io
# Open index.html in your browser
```

## 🌙 Dark Mode

The site includes a built-in dark mode toggle (top-right corner):
- Click the 🌙/☀️ button to switch themes
- Your preference is saved in `localStorage`
- Automatically detects system preference on first visit

## 🛠️ Customization

Edit `index.html` directly — the entire site is a single HTML file with embedded CSS and JS:

- **Colors**: Update CSS custom properties in `:root` and `[data-theme="dark"]`
- **Content**: Modify the HTML sections (hero, stats, projects, links)
- **Avatar**: Change the `src` in the `<img class="avatar">` tag
